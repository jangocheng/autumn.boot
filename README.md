# Autumn.Boot后端轻量级快速开发框架
本框架采用DDD领域驱动的设计模式，将相关功能解耦为可独立部署的站点并整合其他优秀的开源框架，后期根据业务需要配合网关和服务治理可直接搭建为微服务，让后端开发人员专注业务层面可配置化的快速开发。
